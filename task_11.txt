1. INNER JOIN: The INNER JOIN selects records that have matches values in both tables.

The INNER JOIN selects all rows from both the tables as long as there is a match between columns.

If there are records in the Category table that do not have matches in Products table, those data will not be shown.


2. LEFT JOIN: The LEFT JOIN returns all tables in left table (table A) and the matching records from the right table (table B). 

The LEFT JOIN returns all records from left table (table A), even if there are no matches in right table (table B).


3: RIGHT JOIN: The RIGHT JOIN keyword returns all records from the right table (table B) and the matching records from the left table (table A).

The RIGHT JOIN returns all records from right table (table B), even if there are no matches in left table (table A).


4: FULL OUTER JOIN:The FULL OUTER JOIN returns all matching records from both tables whether the other table matches or not. 


5: CROSS JOIN:  In CROSS JOIN we will multiply table A with table B.

Each record in table A will be multiplied with table B.


6: SELF JOIN: SELF JOIN is a type of join that combines the records of a table with itself. 

Self Join only requires one table to join itself.
